What This Demonstrates

Five distinct multi-agent architecture patterns. Each preset uses a different approach to decompose and execute tasks:

Fan-Out — parallel workers, merged result

Router — classify, dispatch to specialist

Map-Reduce — gather in parallel, aggregate

Sequential Chain — ordered steps with gates

ReAct Loop — reason-act-observe cycles

Choosing the right pattern for each task reduces cost, improves reliability, and keeps agent behavior predictable in production.
